# CSDC100_Pre-Finals
A repository holding a websie about GenAI FAQS, and Ethics, to be pass for prefinals in Computing
# Project Title
 CSDC100 Pre-finals Output
# Short overview 
This project is about creating a website containing the general information of Generative AI. This includes what exactly Generative AI (GenAI) is, its weaknesses, the ethical implications, suggested guidelines, risks and responsibilities, and risks. 
# Instructions on how to run the project
1. Download the zip file of the entire repository
2. Extract the zip file to a folder, ensure that all files of the zip folder are all contained in one folder
3. Run index.html
4. Click the links on the website to go through the other webpages
# Frameworks/libraries used
This project uses the Bootstrap 5 library to function. Additionally, it also uses css style formatting for most of the design.
